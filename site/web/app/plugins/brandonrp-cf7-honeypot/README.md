# BrandonRP CF7 Honeypot

Invisible honeypot + timing check for **Contact Form 7**. See `brandonrp-cf7-honeypot.php` for constants.

**Activate:** WP Admin → Plugins → BrandonRP CF7 Honeypot.

Bedrock ignores `web/app/plugins/*` by default; this folder is whitelisted in `site/.gitignore` so it can be committed like `dbsync`.
